
# Simple Interest Calculator

## Description
This project contains a Bash script to calculate simple interest.

## Formula
Simple Interest = (Principal × Rate × Time) / 100

## How to Run
1. Run the script using: bash simple-interest.sh
2. Enter the principal amount
3. Enter the rate of interest
4. Enter the time period

## Example
If Principal = 1000  
Rate = 5  
Time = 2  

Simple Interest = 100
